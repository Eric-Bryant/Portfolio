My Media Lists

This project was built using Vue.js, Vuetify, Firebase (Authentication, Cloud Firestore), tMDB API, and Express.js. With My Media Lists you can create an account and search for movies or TV shows to add to lists you create.

* The backend that handles the API is hosted on Heroku and requires being "woken up" if there hasn't been any traffic recently. This shouldn't take more than a few seconds upon visiting the site.
